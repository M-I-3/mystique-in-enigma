# MIE Portfolio

This is a static portfolio site for MIE.

## Image placement

Place the provided profile image here:

`assets/MIE_Profile_512px.jpg`

## Form setup

The contact form currently points to a placeholder Formspree endpoint:

`https://formspree.io/f/your-id`

Replace `your-id` with your actual Formspree form ID before publishing.
